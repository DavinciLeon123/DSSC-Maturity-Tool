<task type="auto">
  <name>Task 2: Create ContextCallout component</name>
  <files>
    frontend/src/components/questionnaire/ContextCallout.tsx
  </files>
  <action>
    **Create a ContextCallout component that displays explanatory text and/or image content.**

    Props: `{ contextText?: string | null; contextImage?: string | null }`

    Logic:
    - If BOTH contextText and contextImage are null/undefined: render nothing (return null)
    - Otherwise: render a styled callout box above the question list

    Callout box style (matches existing coe-dsc.nl CSS variable pattern):
    - Background: a light tint (e.g., `#F0F4F8` — a light blue-gray, or `var(--color-bg-light)` if defined in globals.css)
    - Border-left: 4px solid `var(--color-navy)` (or `var(--color-green)`)
    - Padding: 1rem 1.25rem
    - Border-radius: `var(--border-radius-sm)`
    - Margin-bottom: 1.5rem

    Content rendering:
    - If contextText is non-null: render as a `<p>` element with `white-space: pre-wrap` to preserve line breaks
    - If contextImage is non-null: render as `<img src={contextImage} alt="Context illustration" style={{ maxWidth: "100%", marginTop: contextText ? "0.75rem" : 0, borderRadius: "var(--border-radius-sm)" }} />`

    Example:
    ```tsx
    export function ContextCallout({ contextText, contextImage }: Props) {
      if (!contextText && !contextImage) return null;
      return (
        <div style={{
          background: "#F0F4F8",
          borderLeft: "4px solid var(--color-navy)",
          padding: "1rem 1.25rem",
          borderRadius: "var(--border-radius-sm)",
          marginBottom: "1.5rem",
        }}>
          {contextText && <p style={{ margin: 0, fontSize: "0.9rem", color: "var(--color-text-gray)", whiteSpace: "pre-wrap" }}>{contextText}</p>}
          {contextImage && <img src={contextImage} alt="Context illustration" style={{ maxWidth: "100%", marginTop: contextText ? "0.75rem" : 0, borderRadius: "var(--border-radius-sm)" }} />}
        </div>
      );
    }
    ```

    Do NOT use antd. Plain styled div matching existing component pattern.
  </action>

    - File exports a named function `ContextCallout`
    - File returns null when both props are null/undefined
    - `cd frontend && npx tsc --noEmit` passes
