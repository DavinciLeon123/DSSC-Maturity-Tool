**Changes (visual only):**

2. Replace raw `<input type="email">` with antd `Form.Item` + `Input`.
3. Replace raw `<input type="password">` with antd `Form.Item` + `Input.Password`.
4. Replace raw `<button type="submit">` with antd `Button type="primary"` with `loading={loading}`.
5. Replace raw error div with antd `Alert type="error"`.
6. Replace raw warning banners (session expired, reset success) with antd `Alert`.
7. Wrap everything in a full-height dark-navy container with a centered white card.

**Key design values (from 07-01 theme + CONTEXT.md):**
- Outer container: `minHeight: '100vh'`, `background: '#06004f'`, `display: 'flex'`, `alignItems: 'center'`, `justifyContent: 'center'`
- Card: `background: 'white'`, `borderRadius: '16px'`, `padding: '2.5rem'`, `width: '100%'`, `maxWidth: '420px'`, `boxShadow: '0 8px 40px rgba(6,0,79,0.15)'`
- Heading: Rubik, 1.5rem, weight 700, color `#06004f`
- Subheading/label: "CoE-DSC / TNO" in green `#399e5a`, small, weight 600
- Submit button: `type="primary"`, `block`, `size="large"`, uses antd `colorPrimary` from ConfigProvider (= `#06004f`)

**onFinish pattern for antd Form (while keeping existing logic):**

```typescript
// Existing handleSubmit stays as-is (it reads from useState e.preventDefault pattern)

```

The simplest migration: keep the `<form onSubmit={handleSubmit}>` wrapper and replace only the inner elements with antd components. This avoids refactoring the submit handler.

**Example inner structure:**

```typescript
import { Form, Input, Button, Alert } from 'antd';

// Inside the form:
<form onSubmit={handleSubmit}>
  {sessionExpired && <Alert message="Your session expired. Please log in again." type="warning" style={{ marginBottom: 16 }} showIcon />}
  {resetSuccess && <Alert message="Password reset successfully. Please log in with your new password." type="success" style={{ marginBottom: 16 }} showIcon />}
  {error && <Alert message={error} type="error" style={{ marginBottom: 16 }} showIcon />}

  <div style={{ marginBottom: '1rem' }}>
    <label style={{ display: 'block', fontSize: '0.875rem', fontWeight: 500, marginBottom: '0.375rem', color: '#06004f', fontFamily: "'Rubik', sans-serif" }}>
      Email
    </label>
    <Input
      type="email"
      value={email}
      onChange={(e) => setEmail(e.target.value)}
      required
      size="large"
    />
  </div>

  <div style={{ marginBottom: '1.5rem' }}>
    <label style={{ display: 'block', fontSize: '0.875rem', fontWeight: 500, marginBottom: '0.375rem', color: '#06004f', fontFamily: "'Rubik', sans-serif" }}>
      Wachtwoord
    </label>
    <Input.Password
      value={password}
      onChange={(e) => setPassword(e.target.value)}
      required
      size="large"
    />
  </div>

  <Button
    type="primary"
    htmlType="submit"
    block
    size="large"
    loading={loading}
    style={{ borderRadius: '8px', height: '48px', fontFamily: "'Rubik', sans-serif", fontWeight: 600 }}
  >
    Inloggen
  </Button>
</form>
```
