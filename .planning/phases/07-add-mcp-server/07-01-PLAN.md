# Plan 07-01: Theme Foundation

## Goal

Establish the antd v6 `ConfigProvider` with Figma-accurate design tokens and load the Rubik font globally. All subsequent plans inherit these tokens automatically ŌĆö no per-screen font or color setup needed.

## Context

- `frontend/src/main.tsx` currently wraps the app in `QueryClientProvider` + `RouterProvider` only. No `ConfigProvider` exists.
- `frontend/src/lib/` directory exists (contains `auth.ts`, `api.ts`, `questionnaire.ts`). Add `theme.ts` here.
- `frontend/index.html` is where the Google Fonts `<link>` tag goes (Vite's entry HTML).
- Design tokens (locked from CONTEXT.md): dark blue `#06004f`, green `#399e5a`, Rubik font, card radius 16px/8px, button blue `#00006b`.

### Wrap app in `ConfigProvider` in `frontend/src/main.tsx`

Edit `frontend/src/main.tsx`:

1. Add imports at the top:
   ```typescript
   import { ConfigProvider } from 'antd';
   import { mamiTheme } from './lib/theme';
   ```

2. Wrap the existing render tree so `ConfigProvider` is the outermost wrapper (outside `QueryClientProvider`):
   ```typescript
   createRoot(document.getElementById('root')!).render(
     <StrictMode>
       <ConfigProvider theme={mamiTheme}>
         <QueryClientProvider client={queryClient}>
           <RouterProvider router={router} />
         </QueryClientProvider>
       </ConfigProvider>
     </StrictMode>,
   )
   ```

**Preserve all existing imports and the router/queryClient setup. Do not remove `StrictMode`.**

**Verify in browser DevTools console after hot-reload:**
```javascript
document.documentElement.style.getPropertyValue('--ant-color-primary')
// Should return '#06004f' (or the MCP-confirmed value)
```

</task>
